Adding a subscriber to the service JustClick when a user registers on the site.

== Description ==

After user registration, an automatic subscription to the specified group in the indicated Justclick account occurs.

= Features =
* Automatic user registration in the “Justclick” system.
* The ability of marker specification for subscribers coming from the website.

== Installation ==

1. Upload `wp_justclick` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
